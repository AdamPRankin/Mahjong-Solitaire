The MahjongStudio is a layout editor created as an extention of the Mahjong game.
MahjongStudio is designed to provide a graphical user interface for creating layouts for the game.

    Assets:
        Assets specific to the MahjongStudio code separated to maintain static assets 
    JS:
        Most JavaScript files are extended from the main game files in order to alter
        the games purpose while maintaining most of the layout logic.
